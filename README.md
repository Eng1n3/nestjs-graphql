# ON BOARD
Aplikasi web untuk users menyimpan data profil dan project yang telah dibuat 

## Requirements System
- nodejs
- nestjs framework
    - graphql
    - postgresql
    - typeorm

## Instalation
Gunakan node package manager npm untuk install aplikasi.
``` bash
npm i
```

## Running development
``` bash
npm run start:dev
```

## Running production
``` bash
npm run start:prod
```
